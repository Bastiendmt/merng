# MERNG (twitter-like)
This is a fullstack project teaching : 
- React
- GraphQL
- Apollo
- Node.js
- MongoDB
- Semantic UI

## Features 
- User 
  - register
  - login
- Posts
  - create 
  - display full post
  - delete
  - like
  - comment
